# 🦦 Mediluna

> Acquire fish. Ignore deadlines.

Mediluna is the unofficial traveling companion of the Boris ecosystem.

She first appeared during OpenAI Build Week while helping guide the development of Boris, a static site compiler written in Zig. What began as a mascot quickly became an optimistic little companion for documentation, open source, and long coding sessions.

---

## Personality

Mediluna is:

- 🦦 endlessly optimistic
- 🎒 always prepared for an adventure
- 🐟 highly motivated by fish
- 🌲 happiest outside
- 💚 convinced every compiler warning can be solved after a snack

She is not a software engineer.

She simply accompanies them.
